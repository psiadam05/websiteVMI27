Text is used by the (p) tag.

Bold is used by the (b) tag.

Italics is used by the (i) tag.

Superscripts are used by the (sup) tag. Ex: sin2(x)+cos2(x)=1

Subscripts are used by the (sub) tag. Ex: loge(x)=ln(x)

Break lines use the
(br /) tag.

Horizontal lines use the

(hr /) tag.

Important information uses the (strong) tag.

Emphasis on information uses the (em) tag.

The (blockquote) tag makes a block of cited text. Ex:

Do you have what it takes?

The (q) tag makes an embeded citation. Ex: Do you have what it takes?

The (abbr) tag makes and defines an abbreviation. Ex: SgtMaj Sowers

The (acronym) tag makes and defines an acryonym. Ex: USSF

The (cite) tag cites a paper or book. Ex: HTML & CSS: design and build websites

The (dfn) tag defines a term. Ex: A derivative finds the rate of change.

The (address) tag defines an address such as location or email. Ex:

serfassap27@vmi.edu

VMI babyyyy

The (ins) tag typically underlines text. Ex: Shannon is the best

The (del) tag typically crosses text. Ex: Shannon is the worst

The (s) tag crosses out text like (del). Ex: Shannon is my girlfriend wife

Lists

The (ol) tag creates an ordered list using the (li) tag for elements. Ex: Process of integration:

  1. U-sub
  2. Parts
  3. Trig
  4. Partial Fraction

The (ul) tag creates an unordered list using the (li) tag for elements. Ex Why Shannon's perfect:

  • Smarts
  • Morals
  • Looks
  • MultiLingual
  • Loyal

The (dl) tag creates a definition list using (dt) for terms and (dd) for definitions. Ex:

Shannon
Perfection if it was a girl

The (a) tag creates a link in text. Ex: VMI

The (mailto:) part of the href atribute is for emails (see (address) tag).

The (target) attribute that comes after href is for opening in a new window. Ex: VMI
